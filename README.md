# Notes

These are class notes, code snippets, random thoughts, resource lists. You can contribute by Forking the repo.

Published at https://dash.training/notes

All files need to include this at the top, where:

* layout is always **notes**
* tags can be anything
* categories should align with existing content

```
---
layout: notes
tags: [<tag1>, <tag2>]
categories: [PowerShell | Azure | PKI]
---
```
